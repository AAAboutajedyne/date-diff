# Welcome to Date Diff App

## Getting Started

First, intall & run the api server (Back-end part) 
```bash
$ cd date-diff-apis
$ npm install

  $ npm run dev
Or
  $ npm run build
  $ npm start
```

Second, intall & run the front-end app 
```bash
$ cd date-diff-app
$ npm install

  $ npm run dev
Or
  $ npm run build
  $ npm start
```

Open http://localhost:3000 with your browser to see the result.
